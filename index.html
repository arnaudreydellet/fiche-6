Trouver un dénominateur commun (2)
La règle
Quand aucun des deux dénominateurs n'est dans la table de l'autre, on ne peut pas garder l'un des deux.

La solution : on multiplie les deux dénominateurs entre eux pour obtenir le dénominateur commun.

Ensuite, on adapte chaque numérateur en le multipliant par le dénominateur de l'autre fraction.

C'est exactement pareil pour une soustraction : on suit les mêmes étapes, puis on soustrait les numérateurs.
